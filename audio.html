Lossless and lossy Audio


Lossless Audio Files

Examples of Lossless File types:

  • Images- PNG, BMP and Raw
  • Audio- OOG, MP4 and MP3

Why use Lossless file formats?

Lossless file formats are data compression alogrithms that keep the original data perfectly reconstructed and identical when files are transferred


Lossy Audio Files

Examples of Lossy File Types:

  • Images- Gif, and JPEG
  • Audio- OGG,MP4,MP3

    • How does Lossy compression work?

    Lossy compression decreases the file size, eliminating unnecessary information making the file smaller

    Why use Lossy File Fomats?

    Lossy files are necessary when sending files decreasing the size,deleting unecessary data
